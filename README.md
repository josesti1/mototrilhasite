# MotoTrilha — v33 (Tailwind + CJS)

## Rodar
1) npm install
2) npm run dev  → web 5173 / api 3001

## Coloque as mídias em /public
- logo.jpeg, flayer.jpeg, parceiro1.png
- foto1.jpeg ... foto10.jpeg
- vídeos: coloque em /public e atualize nomes no App.tsx se necessário

- Tailwind + PostCSS configurados (postcss.config.js, tailwind.config.js, src/styles.css).
